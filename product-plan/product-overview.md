# Dominitec — Product Overview

## Summary

A polished, professional landing page for Dominitec, a software development company based in the Dominican Republic. The site promotes custom software and web/mobile app development services to startups and small businesses, emphasizing cost-effective quality, bilingual capabilities, and fast delivery.

## Planned Sections

1. **Hero** — Eye-catching introduction with Dominitec's value proposition and a clear call-to-action.
2. **Services** — Brief overview of custom software and web/mobile app development offerings.
3. **Contact** — Get a Quote form that converts visitors into leads.

## Data Model

No global data model — this is a static landing page with section-specific content.

## Design System

**Colors:**
- Primary: `blue` — Used for buttons, links, key accents
- Secondary: `red` — Used for tags, highlights, secondary elements
- Neutral: `slate` — Used for backgrounds, text, borders

**Typography:**
- Heading: Inter
- Body: Inter
- Mono: JetBrains Mono

## Implementation Sequence

Build this product in milestones:

1. **Foundation** — Set up design tokens, routing structure, and base layout
2. **Hero** — Eye-catching introduction with value proposition and CTA
3. **Services** — Service cards showcasing offerings
4. **Contact** — Quote request form with validation

Each milestone has a dedicated instruction document in `product-plan/instructions/`.
